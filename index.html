SismoAlerta

Sistema educativo de alerta sísmica diseñado para guiar acciones seguras durante eventos sísmicos.

¿Quiénes se benefician?

Administradores

Pueden supervisar el estado del sistema y asegurar su correcto uso.

Docentes

Cuentan con una guía visual y auditiva para mantener la calma y evacuar correctamente.

Estudiantes

Reciben instrucciones claras para actuar sin pánico y con seguridad.

Características Principales

  • ✅ Activación inmediata de alerta sonora y visual.
  • ✅ Simulaciones para prácticas de evacuación.
  • ✅ Sistema diseñado para instituciones educativas.
  • ✅ Comunicación clara para prevenir el pánico.
